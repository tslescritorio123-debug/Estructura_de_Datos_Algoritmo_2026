🧮 Calculadora de Estructura de Datos - C++:
Este proyecto es una herramienta de consola diseñada para realizar operaciones aritméticas fundamentales. Ha sido desarrollado como parte del curso de Estructura de Datos, aplicando conceptos de lógica de programación y control de flujo en C++.
🛠️ Características Técnicas:
La calculadora implementa una lógica de decisión eficiente mediante una estructura switch-case, lo que garantiza un tiempo de respuesta óptimo.
Tipos de datos: Uso de float para permitir operaciones con decimales y mayor precisión.
Seguridad del Código: Incluye una validación crítica para evitar el error de ejecución por división entre cero.
Interfaz: Interacción clara mediante consola con mensajes dirigidos al usuario.
🚀 Operaciones Disponibles:
Suma+
Resta-
Multiplicación*
División/
📦 Instalación y Ejecución
Para ejecutar este código localmente, asegúrate de tener un compilador de C++ instalado:
Clonar el repositorio:
Bashgit clone https://github.com/tslescritorio123-debug/Estructura_de_Datos_Algoritmo_2026.git
Compilar:
Bashg++ calculadora_estructuras.cpp -o calculadora
Ejecutar:
Bash./calculadora
